API Pencarian ODP

API sederhana untuk mencari ODP (Optical Distribution Point) berdasarkan koordinat.

Endpoint: /search_odp

Mencari ODP dalam radius tertentu dari suatu koordinat dan mengembalikan hasilnya sebagai gambar PNG.

Parameter:

lat: Latitude (required) (default: -3.292481)
lng: Longitude (required) (default: 114.592482)
radius: Radius pencarian dalam meter (default: 500)
coords: Tampilkan koordinat (true/false) (default: true)
name: Tampilkan nama ODP (true/false) (default: true)
max: Maksimal ODP yang ditampilkan (default: 50)
json: Kembalikan hasil dalam format JSON (true/false) (default: false)

Contoh:

/search_odp?lat=-3.292481&lng=114.592482&radius=100

Mencari ODP dalam radius 100m dari koordinat -3.292481, 114.592482.

/search_odp?lat=-3.3219&lng=114.6034&radius=1500&max=100

Mencari ODP dalam radius 1500m dan menampilkan maksimal 100 ODP.

/search_odp?lat=-3.8159&lng=114.7505&radius=500&coords=false

Mencari ODP dalam radius 500m tanpa menampilkan koordinat.

/search_odp?lat=-3.292481&lng=114.592482&radius=100&json=true

Mencari ODP dalam radius 100m dan mengembalikan hasil dalam format JSON.
